# Report R0011 - Datenschutz-Seite Accessibility

**Datum:** 2025-12-06
**Task-IDs:** Ad-hoc Performance/Accessibility Optimierung
**Status:** ERLEDIGT
**Seite:** /datenschutz

---

## Ausgangslage

Die Datenschutz-Seite hatte potenzielle Accessibility-Probleme die in PageSpeed Insights zu Punktabzug fuehren koennten:
- Sections ohne aria-labelledby
- SVG-Icons ohne aria-hidden
- Listen ohne role="list"
- Mobile Modal ohne role="dialog"

---

## Durchgefuehrte Optimierungen

### 1. Section aria-labelledby (14x)

Alle Content-Sections mit `aria-labelledby` und h2 mit IDs versehen:

```tsx
// Vorher:
<section id="ueberblick" className="section-tracked content-block">
  <h2 className="section-title">1. Datenschutz auf einen Blick</h2>

// Nachher:
<section id="ueberblick" className="section-tracked content-block" aria-labelledby="ueberblick-heading">
  <h2 id="ueberblick-heading" className="section-title">1. Datenschutz auf einen Blick</h2>
```

**Betroffene Sections:**
- datenschutz-header
- datenschutz-hauptteil
- ueberblick
- hosting
- allgemeine-hinweise
- datenerfassung
- cookies
- kontaktformular
- soziale-medien
- analyse-tools
- newsletter
- plugins
- kundendaten
- bewerberdaten
- betroffenenrechte
- aenderungen

### 2. SVG aria-hidden (7x)

Dekorative Icons fuer Screenreader versteckt:

```tsx
// Vorher:
<svg width="20" height="20" viewBox="0 0 24 24" ...>

// Nachher:
<svg width="20" height="20" viewBox="0 0 24 24" ... aria-hidden="true">
```

**Betroffene Icons:**
- Download-Button SVGs (2x)
- Mobile Nav Trigger SVG
- Mobile Nav Close SVG
- Scroll-to-Top SVG
- Scroll-to-Bottom SVG

### 3. Listen role="list" (5x)

```tsx
// Vorher:
<ul className="standard-list">

// Nachher:
<ul className="standard-list" role="list" aria-label="...">
  <li role="listitem">
```

**Betroffene Listen:**
- Rechtsgrundlagen der Datenverarbeitung
- Server-Log-Daten
- Verarbeitete Kundendaten
- Ihre Rechte nach DSGVO
- Sidebar Navigation
- Mobile Nav Menu

### 4. Mobile Modal Accessibility

```tsx
// Vorher:
<div className="mobile-nav-modal ...">

// Nachher:
<div
  id="mobile-nav-dialog"
  className="mobile-nav-modal ..."
  role="dialog"
  aria-modal="true"
  aria-labelledby="mobile-nav-title"
  aria-hidden={!isMobileNavOpen}
>
```

**Mobile Trigger Button:**
```tsx
aria-expanded={isMobileNavOpen}
aria-controls="mobile-nav-dialog"
```

### 5. Sidebar Navigation

```tsx
// Vorher:
<aside className="datenschutz-sidebar">
  <nav>
    <h3 className="sidebar-title">Inhaltsverzeichnis</h3>
    <ul className="sidebar-nav">

// Nachher:
<aside className="datenschutz-sidebar" aria-label="Seitennavigation">
  <nav aria-label="Inhaltsverzeichnis Datenschutz">
    <h3 className="sidebar-title" id="sidebar-nav-title">Inhaltsverzeichnis</h3>
    <ul className="sidebar-nav" role="list" aria-labelledby="sidebar-nav-title">
```

### 6. Kontrast-Verbesserung (CSS)

```css
/* Vorher: */
--text-body: #64748b; /* gray-500 - zu schwacher Kontrast */

/* Nachher: */
--text-body: #475569; /* gray-600 - WCAG AA konform */
```
