# Report R0012 - AGB-Seite Accessibility

**Datum:** 2025-12-06
**Task-IDs:** Ad-hoc Performance/Accessibility Optimierung
**Status:** ERLEDIGT
**Seite:** /agb

---

## Ausgangslage

Die AGB-Seite hatte dieselben Accessibility-Probleme wie die Datenschutz-Seite:
- Sections ohne aria-labelledby
- SVG-Icons ohne aria-hidden
- Listen ohne role="list"
- Mobile Modal ohne role="dialog"
- Kontrast zu schwach (gray-500)

---

## Durchgefuehrte Optimierungen

### 1. Section aria-labelledby (4x Hauptsections)

```tsx
// Vorher:
<section id="agb-hauptteil" className="section-tracked content-block">
  <h2 className="section-title">Teil A: Allgemeine Geschäftsbedingungen</h2>

// Nachher:
<section id="agb-hauptteil" className="section-tracked content-block" aria-labelledby="agb-hauptteil-heading">
  <h2 id="agb-hauptteil-heading" className="section-title">Teil A: Allgemeine Geschäftsbedingungen</h2>
```

**Betroffene Sections:**
- agb-header (Seiten-Titel)
- agb-hauptteil (Teil A)
- rechtliche-hinweise (Teil B)
- unternehmenshinweise (Teil C)

### 2. SVG aria-hidden (4x)

```tsx
// Vorher:
<svg xmlns="http://www.w3.org/2000/svg" viewBox="0 0 24 24" ...>

// Nachher:
<svg xmlns="http://www.w3.org/2000/svg" viewBox="0 0 24 24" ... aria-hidden="true">
```

**Betroffene Icons:**
- Mobile Nav Trigger SVG
- Mobile Nav Close SVG
- Scroll-to-Top SVG
- Scroll-to-Bottom SVG

### 3. Listen role="list" (10x)

```tsx
// Vorher:
<ul className="standard-list">

// Nachher:
<ul className="standard-list" role="list" aria-label="...">
  <li role="listitem">
```

**Betroffene Listen:**
- Leistungsbereiche
- Vertragsschluss-Optionen
- Unterhaltsreinigung Umfang
- Mitwirkungspflichten
- Verguetungsmodelle
- Kuendigungsgruende
- Handelsregisterstatus
- Geplante Zertifizierungen
- Qualitaetsgrundsaetze
- Sidebar Navigation
- Mobile Nav Menu

### 4. Mobile Modal Accessibility

```tsx
// Nachher:
<div
  id="mobile-nav-dialog"
  className="mobile-nav-modal ..."
  role="dialog"
  aria-modal="true"
  aria-labelledby="mobile-nav-title"
  aria-hidden={!isMobileNavOpen}
>
```

**Mobile Trigger Button:**
```tsx
aria-expanded={isMobileNavOpen}
aria-controls="mobile-nav-dialog"
```

### 5. Sidebar Navigation

```tsx
// Nachher:
<aside className="agb-sidebar" aria-label="Seitennavigation">
  <nav aria-label="Inhaltsverzeichnis AGB">
    <h3 className="sidebar-title" id="sidebar-nav-title">Inhaltsverzeichnis</h3>
    <ul className="sidebar-nav" role="list" aria-labelledby="sidebar-nav-title">
```

### 6. Kontrast-Verbesserung (CSS)

```css
/* Vorher: */
--text-body: #64748b; /* gray-500 */

/* Nachher: */
--text-body: #475569; /* gray-600 - WCAG AA konform */
```
