# P0-07: Rechtliches auditieren

## Gefundene & behobene Probleme

### 1. KRITISCH: Falsche Website-URL im Impressum

| Datei | Zeile | Vorher | Nachher |
|-------|-------|--------|---------|
| `app/impressum/ImpressumContent.tsx` | 168 | `www.fimi-gebaeudereinigung.de` | `www.fimi-service.de` |

### 2. WICHTIG: Inkonsistente Rechtsform (GmbH vs. GmbH i.Gr.)

Da das Handelsregister noch "In Bearbeitung" ist, muss überall "i.Gr." (in Gründung) stehen.

**Korrigierte Stellen (16 Änderungen in 7 Dateien):**

| Datei | Änderung |
|-------|----------|
| `app/impressum/page.tsx` | Metadata: title + description |
| `app/impressum/ImpressumContent.tsx` | Firmenname (3x) |
| `app/datenschutz/page.tsx` | Metadata: title + description + Umlaute korrigiert |
| `app/datenschutz/DatenschutzContent.tsx` | Header-Subtitle + Verantwortliche Stelle |
| `app/agb/page.tsx` | Metadata: title + description + Umlaute korrigiert |
| `app/agb/AGBContent.tsx` | Header + Geltungsbereich + Gültigkeit + Handelsregister (4x) |
| `components/GoogleMapsWrapper.tsx` | Firmenname im Consent-Overlay |

### 3. AGB-Link korrigiert

| Datei | Zeile | Vorher | Nachher |
|-------|-------|--------|---------|
| `app/agb/AGBContent.tsx` | 478 | `fimi-reinigung.ch/datenschutz` | `fimi-service.de/datenschutz` |

### 4. NEU: Rechtlicher Disclaimer für Partner-/Lieferantenlogos

Die PartnerLogosSlider-Komponente zeigt 13 Marken (Kärcher, Unger, Vermop, etc.). Um keine falschen Partnerschaften/Kooperationen zu suggerieren, wurde ein Disclaimer hinzugefügt:

**Datei:** `components/PartnerLogosSlider.tsx` (Zeile 284-289)

```
Hinweis: Die aufgeführten Marken und Hersteller dienen ausschließlich der
Information über von uns eingesetzte oder potenziell einsetzbare Produkte
und Geräte. Es besteht keine Geschäftspartnerschaft, Kooperation oder
Exklusivvereinbarung mit den genannten Unternehmen. Der Einsatz erfolgt
projektbezogen je nach Anforderung und Verfügbarkeit.
```
