## Fix 1: PartnerLogosSlider - DOM-Duplikate (P0-19)

**Problem:**
```jsx
{[...partner, ...partner, ...partner].map((item, index) => (
```
13 Partner × 3 = 39 DOM-Elemente. Screenreader las alle 39 vor.

**Lösung:**
- `aria-hidden="true"` auf den animierten Slider-Container
- Neue `sr-only` Liste mit nur 13 Einträgen für Screenreader

**Datei:** `components/PartnerLogosSlider.tsx`

```tsx
// Vorher:
<motion.div animate={controls} className="flex gap-4 sm:gap-6 py-4">

// Nachher:
<motion.div animate={controls} className="flex gap-4 sm:gap-6 py-4" aria-hidden="true">
  {/* ... 39 visuelle Elemente ... */}
</motion.div>
{/* Screenreader-only: Liste ohne Duplikate */}
<ul className="sr-only" role="list">
  {partner.map((item) => (
    <li key={item.name}>{item.name} - {item.kategorie}</li>
  ))}
</ul>
```

## Fix 2: PartnerLogosSlider - Section aria-label (P0-18)

**Problem:** Section hatte keine Beschreibung für Screenreader.

**Lösung:**
```tsx
// Vorher:
<section className={`py-12 lg:py-16 overflow-hidden ${className}`}>

// Nachher:
<section className={`...`} aria-label="Partner und Hersteller">
```

---

## Fix 3: ProcessContainer - Icons aria-hidden (P0-18)

**Problem:** Dekorative Icons wurden von Screenreadern vorgelesen.

**Lösung:**
```tsx
// Vorher:
<Icon size={24} className="text-white" strokeWidth={1.5} />

// Nachher:
<Icon size={24} className="text-white" strokeWidth={1.5} aria-hidden="true" />
```

---

## Fix 4: ProcessContainer - role="list" (P0-18)

**Problem:** Schritte-Container hatte keine semantische Listen-Rolle.

**Lösung:**
```tsx
// Vorher:
<div className="space-y-6 mb-10">

// Nachher:
<div className="space-y-6 mb-10" role="list" aria-label="Ablauf in 4 Schritten">
  ...
  <div className="flex gap-4 items-start" role="listitem">
```

---

## Fix 5: HeroContainer - Stadt-Buttons role="group" (P0-18)

**Problem:** Button-Gruppe hatte keine semantische Gruppierung.

**Lösung:**
```tsx
// Vorher:
<span className="text-white/70 font-semibold">Standorte:</span>
<div className="flex items-center gap-2">

// Nachher:
<span className="text-white/70 font-semibold" id="standorte-label">Standorte:</span>
<div className="flex items-center gap-2" role="group" aria-labelledby="standorte-label">
```
